## ğŸ“‹ Project Overview

This project uses **ESP32** to create a **web-based control panel** for monitoring **temperature and humidity** from a **DHT22 sensor** and controlling an **LED**.  
The readings are displayed on a **16x2 I2C LCD** and accessible via a web browser.

**Features:**
- Read **temperature & humidity** from DHT22
- Display readings on **LCD**
- Access data and control LED through **web interface**
- LED control: ON / OFF buttons
- Auto-refresh temperature page every 5 seconds
